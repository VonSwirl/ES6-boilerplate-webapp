# Quick ES6 + NodeJS + ExpressJS + PUG + MVC + 

## Follow Steps Below

1. Create New `.env` file in root
Add these values to it:

      | Values                 | 
      | -------------          |
      | LOCAL_DIRNAME=         |
      | PORT=3000              |
      | NODE_ENV='development' |





2. Run Cmd

      `npm i`




## When Complete Run Cmd To Launch


  `nodemon --exec babel-node app.js`
