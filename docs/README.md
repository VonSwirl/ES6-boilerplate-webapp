# ES6/NodeJS/ExpressJS/PUG/MVC Boilerplate Web-App 

## Installation

  1. Create New `.env` file in root DIR & copy and pastes the values listed below:

        | Values                 | 
        | -------------          |
        | LOCAL_DIRNAME=         |
        | PORT=                  |
        | NODE_ENV='development' |
        | DB_STAGING=            |
        | DB_PRODUCTION=         |
        | DB_DEVELOPMENT=        |
        |                        |

  2. Run:
  ## 
    npm i
----

## Launch in Dev Env:
    nodemon --exec babel-node app.js


## Launch in Prod Env: 
    npm start
